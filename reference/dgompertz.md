# Gompertz Distribution Density Function

Compute the probability density function of a Gompertz distribution.

## Usage

``` r
dgompertz(x, b, eta, log = FALSE)
```

## Arguments

- x:

  Numeric vector. Values at which to evaluate the density.

- b:

  Numeric. Shape parameter (b \> 0).

- eta:

  Numeric. Rate parameter (eta \> 0).

- log:

  Logical. If TRUE, return log density.

## Value

Numeric vector of density values.
