# Estimate Prior Distribution for the Shedding Ratio (zeta_ratio = zeta_1 / zeta_2)

Fits a lognormal prior for the per-person-per-day symptomatic-to-
asymptomatic shedding ratio \\\zeta\_{\mathrm{ratio}} = \zeta_1 /
\zeta_2\\ used in
[`sample_parameters()`](https://institutefordiseasemodeling.github.io/MOSAIC-pkg/reference/sample_parameters.md)
to derive \\\zeta_2 = \zeta_1 / \zeta\_{\mathrm{ratio}}\\ at sampling
time (which guarantees \\\zeta_1 \> \zeta_2\\ algebraically).

## Usage

``` r
est_zeta_ratio_prior(
  PATHS,
  zeta_1_fit = NULL,
  zeta_2_fit = NULL,
  n_sim = 100000L,
  seed = 1L,
  diagnostic_candidate_fits = FALSE
)
```

## Arguments

- PATHS:

  A list of paths generated by
  [`get_paths()`](https://institutefordiseasemodeling.github.io/MOSAIC-pkg/reference/get_paths.md).
  Must include:

  - **MODEL_INPUT**: directory where parameter CSVs are saved.

  - **DOCS_FIGURES**: directory where the summary figure is saved.

- zeta_1_fit:

  Either `NULL`, the full list returned by
  [`est_zeta_1_prior()`](https://institutefordiseasemodeling.github.io/MOSAIC-pkg/reference/est_zeta_1_prior.md),
  or the `$fit` sublist with at least `meanlog` and `sdlog`. If `NULL`,
  `est_zeta_1_prior(PATHS)` is called internally.

- zeta_2_fit:

  Same contract as `zeta_1_fit`; calls `est_zeta_2_prior(PATHS)` if
  `NULL`.

- n_sim:

  Integer. Monte-Carlo sample size for the derived channel. Default
  `100000L`.

- seed:

  Integer. Random-number seed for reproducible paired-bootstrap MC.
  Default `1L`. Byte-equality of the written `priors_default.json`
  across builds requires an identical seed.

- diagnostic_candidate_fits:

  Logical. If `TRUE`, attempt skew-normal / Student-t /
  generalised-normal diagnostic fits via `fitdistrplus`. Default
  `FALSE`. Requires the optional `fitdistrplus` package; if the package
  is not installed a message is emitted and
  `$diagnostics$candidate_fits` remains `NULL`.

## Value

Invisibly returns a list with elements:

- `data`: direct-literature anchor table (Table 7.A).

- `fit`: combined (C) lognormal parameters (`meanlog`, `sdlog`,
  `median`, `mode`, `mean`, `ci_lower`, `ci_upper`).

- `param_df`: long-format parameter data frame (as saved to disk).

- `prediction`: density grid for the direct, derived and combined
  channels (tagged via the `channel` column).

- `diagnostics`: a list with `fit_direct`, `fit_derived`,
  `fit_combined`, `ratio_sample` (paired-bootstrap MC draws on the log10
  scale), `analytic_check` (independent-lognormal closed form,
  cross-check only), and `candidate_fits`.

## Details

The function produces THREE candidate lognormal fits and writes the
precision-weighted combined prior:

- **(A) Direct channel**: weighted-MLE lognormal on literature ratio
  anchors (Smith 2026 medRxiv, Nelson 2009 paired, Kaper 1995, Harris
  2012, Chao 2011, Finger 2018, Righetto 2012, Sugimoto 2014, MOSAIC
  calibration).

- **(B) Derived channel**: paired Monte-Carlo lognormal from the fitted
  \\\zeta_1\\ / \\\zeta_2\\ marginals, with a paired-bootstrap step on
  the shared Nelson 2009 anchor.

- **(C) Combined**: precision-weighted (inverse-variance) lognormal
  combination of (A) and (B). This is what is written into
  `priors_default$parameters_global$zeta_ratio`.

**Direct channel (A).** Weighted MLE on log10 of the literature ratio
anchors in the direct-ratio table. The anchors span ~1.6x (Smith 2026
household per-person-day) to ~10^5 (Nelson 2009 paired stool
concentrations). This spread reflects genuinely different quantities
(household transmission probability vs stool concentration ratio) and is
preserved in the fit's `sdlog`.

**Derived channel (B).** If \\\zeta_1 \sim \mathrm{LN}(\mu_1,
\sigma_1^2)\\ and \\\zeta_2 \sim \mathrm{LN}(\mu_2, \sigma_2^2)\\ are
independent, their ratio is exactly \\\mathrm{LN}(\mu_1 - \mu_2,
\sigma_1^2 + \sigma_2^2)\\. The derived channel uses this analytic
closed form directly. Nelson 2009 is a shared primary source for both
marginals, which would in principle induce positive correlation and
reduce the ratio's variance, but correctly propagating that requires
decomposing each marginal's variance into a Nelson-attributable
component and a residual independent component – a subjective
attribution that is not well-supported by the available literature. The
closed-form variance \\\sigma_1^2 + \sigma_2^2\\ is therefore a
conservative upper bound on the true derived-channel spread. A
100,000-draw Monte Carlo sample of the ratio distribution is generated
for the diagnostic figure and returned in `$diagnostics$ratio_sample`;
it does not enter the fit statistics.

**Combined channel (C).** Given direct and derived fits
\\\mathrm{LN}(\mu_A, \sigma_A^2)\\ and \\\mathrm{LN}(\mu_B,
\sigma_B^2)\\, the precision-weighted mean on the log scale is \$\$\mu_C
= \frac{\mu_A / \sigma_A^2 + \mu_B / \sigma_B^2}{1/\sigma_A^2 +
1/\sigma_B^2}, \quad \sigma_C = \left(1/\sigma_A^2 +
1/\sigma_B^2\right)^{-1/2}\$\$ This is the Bayesian posterior for the
log-ratio under flat priors when both channels are treated as
independent normal likelihoods.

**Smith 2026 dose-response note.** Smith et al. 2026 (medRxiv) report an
odds ratio of 0.60 (95 percent CI 0.11-3.23) for household transmission
with asymptomatic vs symptomatic index cases, giving an inverse
transmission ratio of ~1.6. Under a near-linear Hill dose-response
regime and equal contact patterns this is compatible with a low
per-person-per-day `zeta_ratio`. The value is included at weight 0.5
alongside Nelson's 10^5 stool-concentration ratio so the reader can see
the tension.

**fitdistrplus dependency.** This function does NOT import
`fitdistrplus`. If `diagnostic_candidate_fits = TRUE` the package is
loaded via [`requireNamespace()`](https://rdrr.io/r/base/ns-load.html)
with a graceful fallback; this follows the pattern at
`R/est_initial_R.R:704-710`.
