# Get Proportion of Suspected Cholera Cases and Save Parameter Data Frame

This function generates the parameters for the proportion of suspected
cholera cases (rho) and saves it as a parameter data frame.

## Usage

``` r
get_suspected_cases(PATHS)
```

## Arguments

- PATHS:

  A list containing paths where the parameter data will be saved.
  Typically generated by the
  [`get_paths()`](https://institutefordiseasemodeling.github.io/MOSAIC-pkg/reference/get_paths.md)
  function and should include:

  - **MODEL_INPUT**: Path to the directory where the parameter data
    frame will be saved.

## Value

A data frame containing the parameter values for the proportion of
suspected cholera cases (rho).
