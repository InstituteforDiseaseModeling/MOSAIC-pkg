# Get Vaccine Effectiveness Data

This function saves a dataset containing vaccine effectiveness data
extracted from multiple studies. The data is saved as a CSV file.

## Usage

``` r
get_vaccine_effectiveness_data(PATHS)
```

## Arguments

- PATHS:

  A list containing the paths where the data will be saved. Typically
  generated by the
  [`get_paths()`](https://institutefordiseasemodeling.github.io/MOSAIC-pkg/reference/get_paths.md)
  function and should include:

  - **DATA_VACCINE_EFFECTIVENESS**: Path to the directory where the
    vaccine effectiveness data will be saved.
