# ISO3 Country Codes for All African Countries

A character vector containing the ISO3 country codes for all 54
countries in Africa.

## Usage

``` r
iso_codes_africa
```

## Format

A character vector of ISO3 country codes.

## Examples

``` r
# Access ISO3 codes for all African countries
iso_codes_africa
```
