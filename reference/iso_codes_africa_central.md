# ISO3 Country Codes for Central African Countries

A character vector containing the ISO3 country codes for 8 countries in
Central Africa.

## Usage

``` r
iso_codes_africa_central
```

## Format

A character vector of ISO3 country codes.

## Examples

``` r
# Access ISO3 codes for Central African countries
iso_codes_africa_central
#> [1] "AGO" "CAF" "COD" "COG" "GAB" "GNQ" "STP" "TCD"
```
