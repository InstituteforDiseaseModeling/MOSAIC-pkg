# ISO3 Country Codes for North African Countries

A character vector containing the ISO3 country codes for 6 countries in
North Africa.

## Usage

``` r
iso_codes_africa_north
```

## Format

A character vector of ISO3 country codes.

## Examples

``` r
# Access ISO3 codes for North African countries
iso_codes_africa_north
#> [1] "DZA" "EGY" "LBY" "MAR" "SDN" "TUN"
```
