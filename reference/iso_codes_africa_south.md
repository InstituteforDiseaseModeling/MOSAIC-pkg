# ISO3 Country Codes for Southern African Countries

A character vector containing the ISO3 country codes for 7 countries in
Southern Africa.

## Usage

``` r
iso_codes_africa_south
```

## Format

A character vector of ISO3 country codes.

## Examples

``` r
# Access ISO3 codes for Southern African countries
iso_codes_africa_south
#> [1] "BWA" "LSO" "NAM" "SWZ" "ZAF" "ZMB" "ZWE"
```
