# ISO3 Country Codes for West African Countries

A character vector containing the ISO3 country codes for 16 countries in
West Africa.

## Usage

``` r
iso_codes_africa_west
```

## Format

A character vector of ISO3 country codes.

## Examples

``` r
# Access ISO3 codes for West African countries
iso_codes_africa_west
#>  [1] "BEN" "BFA" "CIV" "CPV" "GHA" "GIN" "GMB" "GNB" "LBR" "MLI" "MRT" "NER"
#> [13] "NGA" "SEN" "SLE" "TGO"
```
