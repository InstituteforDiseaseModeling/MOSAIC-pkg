# ISO3 Country Codes for MOSAIC Modeling Framework

A character vector containing the ISO3 country codes for 41 countries
modeled in the MOSAIC framework.

## Usage

``` r
iso_codes_mosaic
```

## Format

A character vector of ISO3 country codes.

## Examples

``` r
# Access ISO3 codes for countries modeled in the MOSAIC framework
iso_codes_mosaic
#>  [1] "AGO" "BDI" "BEN" "BFA" "BWA" "CAF" "CIV" "CMR" "COD" "COG" "ERI" "ETH"
#> [13] "GAB" "GHA" "GIN" "GMB" "GNB" "GNQ" "KEN" "LBR" "MLI" "MOZ" "MRT" "MWI"
#> [25] "NAM" "NER" "NGA" "RWA" "SEN" "SLE" "SOM" "SSD" "SWZ" "TCD" "TGO" "TZA"
#> [37] "UGA" "ZAF" "ZMB" "ZWE"
```
