# ISO3 Country Codes for Sub-Saharan African Countries

A character vector containing the ISO3 country codes for 48 countries in
Sub-Saharan Africa.

## Usage

``` r
iso_codes_ssa
```

## Format

A character vector of ISO3 country codes.

## Examples

``` r
# Access ISO3 codes for Sub-Saharan African countries
iso_codes_ssa
```
