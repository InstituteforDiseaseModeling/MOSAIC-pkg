# ISO3 Country Codes for WHO AFRO Region Countries

A character vector containing the ISO3 country codes for 47 countries in
the WHO African Region (AFRO).

## Usage

``` r
iso_codes_who_afro
```

## Format

A character vector of ISO3 country codes.

## Examples

``` r
# Access ISO3 codes for WHO AFRO Region countries
iso_codes_who_afro
```
