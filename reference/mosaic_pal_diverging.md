# Generate a Diverging MOSAIC Palette

Produces a palette with a neutral midpoint and symmetric endpoints.

## Usage

``` r
mosaic_pal_diverging(n, palette = "blue_red", direction = 1)
```

## Arguments

- n:

  Number of colors.

- palette:

  Palette name: `"blue_red"` (default), `"blue_orange"`,
  `"green_purple"`.

- direction:

  `1` (default) or `-1` (reversed).

## Value

Character vector of hex color codes.
