# Plot Africa Map with Cholera Outbreak Countries

This function creates a map of Africa highlighting countries in
Sub-Saharan Africa and those with cholera outbreaks in the past 5 and 10
years. The map is saved as a PNG file in the specified
`PATHS$DOCS_FIGURES` directory.

## Usage

``` r
plot_africa_map(PATHS)
```

## Arguments

- PATHS:

  A list containing the paths where the figure will be saved. Typically
  generated by the
  [`get_paths()`](https://institutefordiseasemodeling.github.io/MOSAIC-pkg/reference/get_paths.md)
  function and should include:

  - **DOCS_FIGURES**: Path to the directory where the map image will be
    saved.

## Value

A ggplot object showing the map.

## Details

The function generates a map of Africa, highlighting Sub-Saharan African
countries and those with cholera outbreaks in the past 5 and 10 years.
The map is saved as a PNG file.

## Examples

``` r
if (FALSE) { # \dontrun{
# Assuming PATHS is generated from get_paths()
PATHS <- get_paths()
plot_africa_map(PATHS)
} # }
```
