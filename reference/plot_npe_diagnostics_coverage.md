# Plot NPE Coverage Diagnostics

Creates coverage diagnostic plots for NPE models

## Usage

``` r
plot_npe_diagnostics_coverage(
  diagnostics,
  plots_dir = NULL,
  max_params_per_page = 40,
  verbose = TRUE
)
```

## Arguments

- diagnostics:

  Diagnostics object or path to diagnostics directory

- plots_dir:

  Directory to save plots (optional)

- max_params_per_page:

  Maximum parameters per page (default 40)

- verbose:

  Print progress messages
