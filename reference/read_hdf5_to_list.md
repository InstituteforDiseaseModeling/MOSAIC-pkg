# Read an HDF5 File into an R List

Reads an HDF5 file (supports .h5 or .h5.gz formats) into a named R list.

## Usage

``` r
read_hdf5_to_list(file_path)
```

## Arguments

- file_path:

  Path to the .h5 or .h5.gz file.

## Value

A named R list representing the hierarchical structure of the HDF5 file.
